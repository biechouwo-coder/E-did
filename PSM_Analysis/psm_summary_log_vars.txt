
======================================================================
PSM倾向得分匹配分析报告（对数控制变量版本）
======================================================================

重要说明：
-----------
本分析使用treat变量进行PSM匹配，匹配的是"身份"而非"状态"

• treat: 静态分组变量（Who）
  - treat=1: 在低碳试点名单中的城市（无论何时开始实施）
  - treat=0: 从未在试点名单中的城市
  - treat不随时间变化

• DID: 动态处理变量（When）
  - DID=1: 该年已实施政策（年份 >= 该城市生效年份）
  - DID=0: 该年未实施政策
  - DID随时间变化

一、分析设定
-----------
• 基期年份: 2009年（政策实施前）
• 匹配变量: treat（静态分组变量）
• 控制变量（对数形式）: ln_pgdp, ln_pop_density, ln_industrial_advanced, ln_fdi_openness
  1. ln_pgdp - ln人均GDP
  2. ln_pop_density - ln人口密度
  3. ln_industrial_advanced - ln产业高级化
  4. ln_fdi_openness - ln_FDI开放度
• 卡尺设定: 0.05
• 匹配方法: 1:1最近邻匹配

二、匹配结果
-----------
• 处理组数量 (treat=1): 98 个城市
• 对照组数量 (treat=0): 105 个城市
• 成功匹配: 90 对
• 匹配率: 91.8%
• 平均得分差: 0.0046
• 最大得分差: 0.0390

三、平衡性检验
-----------
匹配前标准化差异 (%):
  • ln_pgdp: 58.92%
  • ln_pop_density: 41.15%
  • ln_industrial_advanced: 36.93%
  • ln_fdi_openness: 44.91%

匹配后标准化差异 (%):
  • ln_pgdp: 39.88%
  • ln_pop_density: 9.78%
  • ln_industrial_advanced: -2.58%
  • ln_fdi_openness: 28.20%

四、匹配质量评估
-----------
• 标准化差异 < 10% 的变量: 2/4
• 结论: 匹配质量一般，半数变量达到平衡标准

五、输出文件
-----------
所有结果已保存至目录: PSM_Analysis/

2. matched_pairs_log_vars.xlsx - 匹配对详细信息

六、后续DID分析建议
-----------
使用匹配后数据集 (matched_dataset_log_vars.xlsx) 进行DID分析：

变量说明：
• treat: 原始分组变量（是否在试点名单中）
• matched_treatment: 匹配后处理组标识（1=匹配成功的试点城市）
• DID: 原始DID变量（是否已实施政策）
• DID_matched: 匹配后DID变量（仅对匹配成功的处理组有效）

推荐模型：
Y = β₀ + β₁×DID_matched + β₂×matched_treatment + γ×控制变量 + δ×年份固定效应 + θ×城市固定效应 + ε

其中：
• DID_matched: 核心解释变量（政策效应）
• matched_treatment: 处理组固定效应
• 控制变量（原始形式）: gdp_per_capita, pop_density, industrial_upgrading, fdi_openness
• 样本期：2007-2023年
• 样本城市：180 个（90对）
